# Resume Builder

A modern Resume Builder built using HTML, CSS, and JavaScript.

## Features

- Live Resume Preview
- Profile Photo Upload
- Education Section
- Experience Section
- Skills
- Projects
- Certifications
- GitHub & LinkedIn Links
- Dark Mode
- Local Storage
- Download Resume as PDF

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- html2canvas
- jsPDF

## How to Run

1. Download the project.
2. Open the folder in VS Code.
3. Run with Live Server.
